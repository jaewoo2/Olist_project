### 4.2 볼레토 (Boleto Bancário) 이해

**볼레토란?**
```
정의: 브라질 은행 이체 결제 시스템
방식: 
1. 주문 시 바코드 생성
2. 고객이 은행/편의점/ATM에서 현금 납부
3. 1-3일 후 판매자 확인

누가 쓰나?
- 신용카드 없는 사람 (20-30%)
- 할부 부담스러운 사람
- 은행 계좌 없는 사람
- 현금 선호자
```

**볼레토 vs 신용카드:**

| 항목 | 볼레토 | 신용카드 |
|------|--------|----------|
| 소득 | 낮음 (중하위층) | 높음 (중상위층) |
| 즉시성 | 느림 (1-3일) | 빠름 (즉시) |
| 할부 | 불가 | 가능 (10-12개월) |
| 취소율 | 높음 (15%) | 낮음 (3%) |
| 수수료 | 낮음 (1-2%) | 높음 (3-5%) |

**분석 시사점:**
```
결제 방법 = 소득 수준 프록시!

볼레토 사용자:
→ 가격 민감
→ 저객단가 예상
→ 할인 프로모션 효과적

신용카드 사용자:
→ 소득 높음
→ 고객단가 가능
→ 편의성, 배송 속도 중시
```

**CRITICAL: 시간 왜곡 방지!**

```
분석 데이터: 2016-2018 (Kaggle)
현재 상황: 2024-2025

절대 섞지 말 것:
❌ "Olist는 유니콘이므로 성장 중"
   → 2021년 이후 이야기

❌ "PIX로 결제 편리"
   → 2020년 도입, 데이터에 없음

❌ "Shopee와 경쟁 치열"
   → 2021년 이후 진입

✅ 올바른 표현:
"2016-2018 당시 Olist는 <1% 점유율의
 작은 마켓플레이스 연결자였으며..."

"현재(2024)는 유니콘이지만,
 분석 시점에는 초기 단계였음"
```

### 12.2 컨텍스트 설정

**프로젝트 프레이밍:**

```

장점:
✅ 시점 일치
✅ 현실적
✅ 데이터 기반

Option B: 비교 분석
"2016-2018 문제가
 2024년에는 해결됐을까?"

필요:
⚠️ 추가 데이터 (Reclame Aqui 등)
⚠️ 시간 많이 걸림
```

### 12.3 차별화 포인트

**일반 분석 vs 우리 분석:**

```
일반적 Olist 분석:
× "고객 세그먼트 발견!"
× "추천 시스템 만들기"
× "매출 예측 모델"

우리 분석 (차별화):
✓ "작은 플레이어의 생존 전략"
✓ "거대 기업 vs 틈새 플레이어"
✓ "한정된 리소스로 최대 효과"
✓ "2018 → 2021 유니콘 경로 탐색"

핵심:
→ 비즈니스 맥락 이해
→ 경쟁 환경 고려
→ 현실적 제약 반영
```

```
발표 시:

 데이터 분석이 전략으로 이어진 사례죠!"

→ 인사이트의 가치 증명!
```
